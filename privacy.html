Privacy Policy

Last updated: June 2026

This Privacy Policy describes how the CricketPOVV automated content distribution tool ("the Bot"), operated by Barinder Saini, handles data in connection with the @CricketPOVV channel and associated social media accounts.

1. Data Collected

The Bot accesses:

  • Video files stored in a private Google Drive folder owned by the account holder
  • Authentication tokens for YouTube, Instagram, and TikTok, used solely to publish content on the account holder's own channels
  • Basic account metadata (e.g., username, account ID) required by each platform's API to publish content

2. Data Use

Collected data is used exclusively to automate the posting of original video content to the account holder's own social media accounts. No data is sold, shared, or disclosed to third parties.

3. Data Storage

Authentication credentials are stored securely on a private server controlled by the account holder and are not accessible to any third party.

4. Third-Party Services

The Bot interacts with the YouTube Data API, Instagram Graph API, TikTok Content Posting API, and Anthropic's Claude API (for AI-generated titles and captions). Each of these services has its own privacy policy governing their handling of data.

5. User Rights

As this tool operates only on accounts owned by the account holder, no end-user data from third parties is collected or processed.
